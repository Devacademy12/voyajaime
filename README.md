# Voyajaime - Application de planification d'itinéraires touristiques en Tunisie

## 🌍 À propos

Voyajaime est une plateforme qui connecte les touristes avec des prestataires d'excursions en Tunisie. Les touristes peuvent planifier leur voyage (mode assisté ou mode libre) et réserver des excursions, tandis que les prestataires gèrent leurs offres et leurs réservations.

**Particularité :** Aucun contact direct entre touristes et prestataires - tout passe par la plateforme.



## 🚀 Installation

```bash
# Cloner le repository
git clone https://github.com/votre-org/voyajaime.git

# Installer les dépendances
cd voyajaime
npm install

# Lancer l'application
npm start
```
